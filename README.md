# astrbot_plugin_random_pic

一个适用于 AstrBot 的随机图片获取插件，支持从配置的多个 API 中随机获取图片，并提供完善的配置与权限管理命令。

---

## ✨ 关键特性

- **随机取图**：从配置的 API 列表中随机抽取 API 获取随机图片。
- **指定取图**：支持指定 API 名称获取对应图片，方便测试与精准获取。
- **多图发送**：支持一次性发送多张图片，并配备白名单与最大数量限制，防止刷屏。
- **智能解析**：自动尝试解析常见 API 返回的 JSON 格式（如 `data[0]["url"]`、`data["link"]` 等），若 API 直接返回图片流或需重定向也能自动处理。
- **在线配置**：管理员可通过指令在线增删 API、管理白名单及调整单次最大获图数量，无需手动修改配置文件。

---

## 📦 前置条件

- 已部署并运行 [AstrBot](https://github.com/Soulter/AstrBot)。
- 插件依赖 `httpx` 库（通常 AstrBot 运行环境中已包含，若缺失请手动安装 `pip install httpx`）。
- 需要可用的随机图片 API（如 LoliconAPI、Manyacg 等）。

---

## 🚀 运行方法

1. 将本插件放置在 AstrBot 的插件目录 `data/plugins/` 下（确保文件夹名为 `astrbot_plugin_random_pic`）;
或使用`git clone https://github.com/YUGANJUN114514/astrbot_plugin_random_pic` 命令将本仓库克隆到插件目录下。
2. 启动或重启 AstrBot。
3. 在 AstrBot 管理面板中启用本插件，插件将会自动加载并生成默认配置。
4. 在聊天环境中发送指令即可使用。

---

## 🛠️ 指令说明

### 基础指令
| 指令 | 说明 |
| :--- | :--- |
| `图图` | 随机从 API 列表中获取一张图片 |
| `多图图 <数量> [API名称]` | 一次获取多张图片（默认使用 manyacg），需白名单权限 |
| `图图测试 指定 <API名称>` | 测试指定名称的 API 是否能正常返回图片 |

### 管理员指令（需 ADMIN 权限）
配置相关指令均以 `图图配置` 开头：

| 指令 | 说明 |
| :--- | :--- |
| `图图配置 白名单 help` | 查看白名单管理帮助 |
| `图图配置 白名单 add <用户ID>` | 将用户加入多图白名单 |
| `图图配置 白名单 remove <用户ID>` | 将用户移出多图白名单 |
| `图图配置 白名单 list` | 查看当前白名单列表 |
| `图图配置 最大数量 <数字>` | 设置 `多图图` 单次最多获取的图片数量（防刷屏） |
| `图图配置 api help` | 查看 API 管理帮助 |
| `图图配置 api add <名称> <API地址>` | 添加新的随机图 API |
| `图图配置 api remove <名称> <API地址>` | 移除指定的 API |
| `图图配置 api list` | 查看当前 API 列表 |

---

## ⚙️ 配置项说明

可在 AstrBot 管理面板的插件配置中直接修改，或使用上述管理员指令动态修改：

- **api**：随机图像 API 列表（字典格式，键为名称，值为地址）。
- **wh_list**：允许使用 `多图图` 指令的用户 ID 白名单。
- **max_num**：`多图图` 单次允许获取的最大图片数量（默认为 5）。
